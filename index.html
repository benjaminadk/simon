INSTRUCTIONS

IMPORTANT - Simon won't let you know if you are wrong until you enter the full amount of presses for each round. The current round is displayed in the box next to the start button

Simon is a memory game. Simon will press a sequence of buttons that will light up. The game begins in round 1, with a 1 button sequence. After Simon does his sequence, you try to match it! If you match it correctly, the game moves to round 2, and Simon does a 2 button sequence. This continues for 20 rounds, then you win. The following explains what to do.

  • Activate ON/OFF Switch. It will light up Orange
  • To play in Regular Mode - In regular mode, when you get a sequence wrong, Simon will replay it and you get another chance.
    • Activate Start Button
  • To play in Strict Mode - In strict mode, when you get a sequence wrong, Simon will turn off and you have to restart the game
    • Activate Strict Button
    • Activate Start Button

HAVE FUN
